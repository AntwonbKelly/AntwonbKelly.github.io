Directory Character and Word Remover

The Directory Remover allows a user to enter a list of characters or a single word, and use these properties to remove strings from multiple Subdirectories.

For example, if i have multiple sub directories all containing the word "Apple" in their name, and i'd like to get rid of the word, i could simply select the main directory hosting the sub directories, enter the desired word into the Remover, and the remover would execute the task
